Twitter Clone

Project Overview

This project is a full-stack clone of Twitter, built using React, Vite, Node.js, Express.js, MongoDB, and Cloudinary for image storage. It provides users with a platform to post tweets, interact with other users, follow/unfollow accounts, and receive real-time notifications.

Technologies Used

  • React
  • Vite
  • Node.js
  • Express.js
  • MongoDB
  • Cloudinary
  • JWT, bcrypt
  • Socket.io

Challenges Faced

  • Real-time updates and notifications using Socket.io
  • Secure JWT authentication and managing user sessions
  • Building a scalable RESTful API with proper middleware
  • Optimizing image storage and retrieval using Cloudinary

Features Implemented

  • User authentication and registration
  • Profile management (users can update their profile picture, bio, etc.)
  • Real-time notifications
  • Follow/unfollow users
  • Posting, editing, and deleting tweets
  • Responsive UI for different devices
