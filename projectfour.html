Project Overview

A full-stack eCommerce price tracker built with Next.js, Bright Data, Cheerio, MongoDB, and Tailwind CSS. It features real-time price tracking, automated web scraping with cron jobs, and email notifications to keep users updated on price changes. The app provides a responsive and intuitive UI.

Technologies Used

  • Next.js
  • Bright Data (Web Scraping)
  • Cheerio
  • MongoDB
  • Tailwind CSS

Challenges Faced

  • Handling large-scale web scraping operations
  • Building a scalable notification system
  • Ensuring data accuracy from multiple sources

Features Implemented

  • Automated price tracking with cron jobs
  • Real-time notifications for price drops
  • Responsive UI for seamless user experience
