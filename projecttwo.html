Project Overview

Developed a machine learning-powered medical recommendation system that predicts diseases based on symptoms and suggests treatments using models like SVC, Random Forest, and Gradient Boosting. Designed and deployed RESTful APIs with Flask, built an interactive web interface with HTML and Bootstrap, and integrated ChromaDB for AI-powered recommendations.

Technologies Used

  • Python
  • Flask
  • PostgreSQL
  • ChromaDB
  • Machine Learning (SVC, Random Forest, Gradient Boosting)

Challenges Faced

  • Ensuring high model accuracy for disease prediction
  • Handling large symptom datasets efficiently
  • Deploying a scalable RESTful API
  • Integrating ChromaDB for AI-driven recommendations

Features Implemented

  • Real-time symptom input and health recommendations
  • Machine learning-based disease prediction
  • Secure API endpoints for retrieving medical data
  • Interactive UI for user-friendly navigation
